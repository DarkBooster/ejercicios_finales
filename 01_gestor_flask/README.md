# 01_gestor_flask

Esqueleto simple de aplicación Flask (con *app factory*).

## Ejecutar

Activá el entorno virtual (ya existe `venv/`) y corré la app:

```bash
cd 01_gestor_flask
source venv/bin/activate
python app.py
```

Sin activar el entorno, también podés ejecutarlo explícitamente con el Python del venv:

```bash
cd 01_gestor_flask
./venv/bin/python app.py
```

Luego abrí `http://127.0.0.1:5000/`.
